This blog is built with Hugo and hosted using GitHub Pages.

The publishing workflow is simple:

1. Write content in Markdown.
2. Commit and push to GitHub.
3. GitHub Actions builds the Hugo site.
4. GitHub Pages publishes the static website.

This gives a free, secure, version-controlled publishing platform for personal branding, technical documentation, and knowledge sharing.
